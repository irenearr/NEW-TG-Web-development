HTML Forms

HTML formulieren zijn de belangrijkste schakel tussen een gebruiker en een website of een applicatie. Zij laten gebruikers toe gegevens naar websites te sturen. Meestal zal dat naar de server zijn maar een webpagina kan de gegevens ook zelf interpreteren. Een HTML form bestaat uit een aantal elementen. Bijvoorbeeld textfields (bestaande uit één of meerdere regels), choices of buttons.

HTML tables

10. Om een tabel te maken heb je de volgende elementen nodig. Geef voor elke element een definitie.
  • <table>: dit element omvat alle andere elementen in de tabel en is nodig om een tabel te maken.
  • <thead>: dit element omvat het deel van de tabel dat de header is, meestal de eerste rij van de tabel.
  • <tbody>: dit element omvat het deel van de tabel dat niet header of footer is, dit element zit in elke tabel ook als het niet expliciet gespecificeerd wordt.
  • <tfoot> dit element omvat het deel van de tabel dat de footer is, bijv een laatste rij waarin de andere rijen worden opgeteld.
  • <th>: table headers, deze cellen staan aan het begin van de rij of kolom en hebben een speciale opmaak.
  • <tr>: table row, eindigt de reeks van cellen zodat de rij wordt afgesloten. Voor een nieuwe rij open je een nieuw tr element en plaats je de td elementen daarbinnen.
  • <td>: table cell, elk td element maakt 1 cel, die naast elkaar komen te staan en zo de eerste rij vormen.
