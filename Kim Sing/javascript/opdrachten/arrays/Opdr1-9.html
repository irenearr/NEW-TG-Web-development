Arrays

  • Push: een item toevoegen aan de array
  • Pop: laatste item verwijderen uit de array
  • Shift: eerste item verwijderen uit de array
  • Slice: selecteert elementen uit een array (die je aangeeft met begin en eind getal), en geeft die weer als een nieuwe array. De nieuwe array begint bij het start argument, maar eindigt voor het eind argument (niet incl). Je moet wel een nieuwe variabele aanmaken voor deze nieuwe array.
  • Splice: adds/removes items to/from an array, and returns the removed item(s). Note: This method changes the original array.
  • Sort: zet de array op (alfabetische) volgorde.
