Opdracht 6

  • child selectors: selecteren alle elementen die "kinderen" zijn van een specifiek element. In tegenstelling tot de descendant selector, die alle elementen die binnen een bepaald element staan selecteert, alle "nakomelingen". Voordelen: .....

    In dit voorbeeld selecteer je alle p elementen die kinderen zijn van een div element:

    div > p {
    color: red;
    }

    Praktijkvoorbeeld child selectors

  • adjacent sibling selectors: selecteren alle elementen die "adjacent" oftewel eerstvolgende "broertjes" zijn van een specifiek element. Voordelen: ....

    In dit voorbeeld selecteer je alle p elementen die eerstvolgende broertjes zijn van div elementen.

    div + p {
    color: red;
    }

    Praktijkvoorbeeld adjacent sibling selectors

  • general sibling selectors: selecteren alle elementen die "broertjes" zijn van een bepaald element. Voordelen: ......

    In onderstaand voorbeeld selecteer je alle p elementen die broertjes zijn van div elementen.

    div ~ p {
    color: red;
    }

    Praktijkvoorbeeld general sibling selectors
