Opdracht 2

CSS pas je toe met een bepaalde syntax. Hoe ziet zo'n syntax eruit? Kun je ook uitleggen wat elk element uit de syntax betekent?

De syntax van CSS bestaat uit een selector en een declaration blok. De selector verwijst naar een HTML element dat wordt gestyled. Het declaration blok bevat een of meerdere declarations gescheiden door puntkomma's. Elke declaration bestaat uit een CSS property name en een waarde, gescheiden door een dubbele punt. Bij meerdere CSS declarations plaats je een puntkomma ertussen, en declaration blocks staan tussen "krulhaakjes".

Voorbeeld:

h1 {
color: blue
}

De selector is hier h1 en de declaration staat tussen krulhaakjes.
