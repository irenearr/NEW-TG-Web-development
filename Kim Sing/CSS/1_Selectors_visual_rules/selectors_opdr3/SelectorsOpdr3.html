Opdracht 3

Er zijn verschillende selectors, bijvoorbeeld;
  • type selectors
  • descendant selectors
  • class selectors
Maak van de bovengenoemde selectors voorbeelden. Geef in je antwoord ook aan wat de voordelen zijn van elke selector.

  • type selectors: deze verwijzen naar alle elementen van een bepaald type, zoals h1, span, em, p, div, etc. Voordeel: uniformiteit?

    h1 {
    color: blue;
    }

  • descendant selectors: selecteert alle elementen die binnen een bepaald type element zitten. Voordeel:.....

    div p {
    background-color: yellow;
    }

  • class selectors: kun je zelf een naam voor verzinnen die je dan in het element plaatst dat je wil stylen. Voordeel: erg specifiek, overtroefd ook bijv de twee bovenstaande selectors.

    .blue {
    color: blue;
    }
