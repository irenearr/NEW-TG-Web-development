Opdracht 1

Je kunt op 3 verschillende manieren CSS in je HTML inladen. Op welke 3 manieren kun je dat doen? Leg ook voor elke wijze uit wat het precies inhoud.
  1. Extern: in een apart css document bij voorkeur in dezelfde map als je html document, gekoppeld via een link in de head sectie van je html document. Het link element vertelt de browser dmv het rel attribute dat het om een styling sheet gaat, en de locatie staat in het href attribute.

    Voorbeeld:

    <head>
    <link rel="stylesheet" href="style.css">
    </head>

  2. Intern: binnenin hetzelfde HTML document, binnen een style element in de head sectie. Nadeel: dit is veel minder flexibel. Je hebt dan op elke pagina een eigen stylesheet, en als je een website maakt met meerdere pagina's worden veranderingen in styling heel veel werk. Gebruik bij voorkeur een externe stylesheet voor uniforme styling tussen webpagina's.

    Voorbeeld:

    <head>
    <style>
    h1 {
    color: blue;
    }
    </style>
    </head>

  3. Inline: zijn alleen van toepassing op een specifiek element en staat in een style attribute. Probeer geen inline styling te gebruiken! Dit is het tegenovergestelde van best practice.

    Voorbeeld:

    <h1 style="color: blue;">Blue heading</h1>

    Blue heading
