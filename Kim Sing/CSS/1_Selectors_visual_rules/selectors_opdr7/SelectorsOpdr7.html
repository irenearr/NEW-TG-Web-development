Maak nu een html bestand aan met voorbeelden van alle soorten basic selectors erin. Maak een link in je antwoord die naar dat html bestand verwijst.
  • type/element selector: verwijst naar een element. Voorbeelden: h1, p, div, input, etc. Praktijkvoorbeeld
  • class selector: selecteert alle elementen met een gegeven class name. Voorbeeld: .class .blue .index etc. Praktijkvoorbeeld
  • ID selector: selecteert een element op basis van zijn id attribute. Elke ID hoort maar bij 1 element, ID's zijn uniek. Voorbeeld: #idname Praktijkvoorbeeld
  • attribute selector: selecteert alle elementen met het gegeven attribute. Voorbeelden: [attr], [autoplay] Praktijkvoorbeeld
  • universal selector: selecteert alle elementen in een document. Voorbeeld: * Praktijkvoorbeeld
